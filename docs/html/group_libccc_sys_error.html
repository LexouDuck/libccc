t_char const* Error_STD_Message(t_errno error)

Returns the error message for the given error

  • Standard library equivalent: (C89) https://en.cppreference.com/w/c/string/byte/strerror

  • Standard library equivalent: (POSIX) https://linux.die.net/man/3/strerror

This is equivalent to the STD C strerror() function (not thread-safe).

Returns:

A global constant string which contains the standard errno message.
