Detailed Documentation

This header defines the common standard math functions for fixed-point numbers.

  • This function is not defined in the C ISO standard.

typedef t_q64 (*f_q64_function)(t_q64 x)

This very small float is typically used to compare two fixed-point values. Since fixed-point types have a configurable fractional portion, it can be useful to have the Fixed_EqualsApprox() functions for approximate equality checks.
