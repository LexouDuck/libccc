◆ list_any

struct list_any

This is a basic linked list struct, with dynamic content type.

The list struct represents one chainlink in the linked-list. So, a linked-list which contains 3 items would consist of 3 different list structs, all chained together via their next pointer, and the last item would have this next pointer set to NULL.
