Detailed Description

This file is simply meant to include all of the basic primitive types.

To be precise, this file exposes the type definitions and macros from:

  • libccc/bool.h
  • libccc/int.h
  • libccc/fixed.h
  • libccc/float.h
  • libccc/pointer.h
  • libccc/char.h
  • libccc/text/ascii.h
  • libccc/text/unicode.h

Note that none of the functions from these headers are included/exposed. If you wish to use the utility functions from a header, you must manually include it: for instance, you must #include <libccc/int.h> to use, say, UInt_ToString().
