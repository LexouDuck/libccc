#define ANSI_STYLE(CODE)    "\x1B["#CODE"m"

ANSI text output display escape code macros.

These are some useful string literals for commandline output colors. Can be used with any of the ‘IO_Output_*’ and ‘IO_Write_*’ functions.

(Learn more here)[https://misc.flogisoft.com/bash/tip_colors_and_formatting]

#define ANSI_STYLE_BLINK    ANSI_STYLE(5)

Display terminal output text which blinks every second (only in tty and xterm)

#define ANSI_STYLE_BOLD ANSI_STYLE(1)

Display terminal output text in bold.

#define ANSI_STYLE_DIM  ANSI_STYLE(2)

Display terminal output text with dimmer, darker colors.

#define ANSI_STYLE_HIDDEN   ANSI_STYLE(8)

Write, but do not display, the terminal output text (useful for passwords)

#define ANSI_STYLE_INVERTED ANSI_STYLE(7)

Display terminal output text with the background/foreground colors reversed.

#define ANSI_STYLE_ITALIC   ANSI_STYLE(3)

Display terminal output text in italic.

#define ANSI_STYLE_OVERLINE ANSI_STYLE(9)

Display terminal output text with an overline (a.k.a strikethrough)

#define ANSI_STYLE_UNDERLINE    ANSI_STYLE(4)

Display terminal output text with an underline.
