Inverse of StringASCII_ToASCIIEscaped

Returns a new null-terminated string where every valid backslash escape sequence is converted to its corresponding string byte. Here is the list of character escape sequences which will be properly parsed:

  • \\ \ (a single backslash, escaping the escape character)
  • ‘&rsquo;Apostrophe -"Double quotes - Bell/Alert/Audible - Backspace -\tTab character -
    Line feed -\vVertical tab -\fForm-feed -\rCarriage return - Escape -\x??Byte value, written as hexadecimal -\u????UTF-8 multi-byte character, written as a hexadecimal code point (Unicode: U+????) -\U????????` UTF-8 multi-byte character, written as a hexadecimal code point (Unicode: U+????????) Any other backslash-ed character will simply resolve to itself (ie: removing the preceding backslash)
Parameters
destThe destination string
strThe string to duplicate, while resolving all escape-sequences to their corresponding char
nThe maximum amount of characters to parse (infinite if 0 is given)
any_escapeIf TRUE, every backslash will be understood as an escape character (ie: any escape sequance will function, with any char after the ‘’\'`)
Returns
The amount of characters parsed from the given str.
