This header defines the common standard math functions and macro defines.

Macros

#define ABS(X)  ((X)<0 ? -(X) :(X))

Computes to the absolute value of X (ie: gives the positive equivalent of X)

  • Standard library equivalent: (C89) https://en.cppreference.com/w/c/numeric/math/abs

  • Standard library equivalent: (C89) https://en.cppreference.com/w/c/numeric/math/fabs

  • Math: \({|x|}\)

#define ISNEG(X)    (!((X)> 0)&&((X) != 0))

Checks if a value is negative (works for unsigned types, without causing tautology warnings)

This macro is useful to check if a value is negative, but its type (signed or unsigned) is not known. In particular, the char and enum native C types are of unknown/undefined signage, so this macro is useful to check that a value is positive, no matter the platform/environment.

  • Math: \({x < 0}\)

#define MAX(X, Y)   ((X)<(Y) ?(Y) :(X))

Expands to the maximum value between X and Y (the larger of the two)

  • Standard library equivalent: (C89) https://en.cppreference.com/w/c/numeric/math/fmax

#define MIN(X, Y)   ((X)<(Y) ?(X) :(Y))

Expands to the minimum value between X and Y (the smaller of the two)

  • Standard library equivalent: (C89) https://en.cppreference.com/w/c/numeric/math/fmin
