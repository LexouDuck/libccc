Detailed Documentation

A union storing a floating-point number of any common size/precision.

These unions are used for certain difficult casting conditions. They are used in particular when casting an <stdarg.h> va_arg to the right type.

Fields

t_f32 fs

floating-point: single precision

t_f64 fd

floating-point: double precision

t_f80 fe

floating-point: extended precision

t_f128 fq

floating-point: quadruple precision
