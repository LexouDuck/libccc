236 /*
237 ** @see StringUTF8_ToEscaped, only additional parameters are documented here
238 **
239 ** If `dest` is NULL, nothing is written and the number of bytes that would be written is returned.
240 ** Even in case of error, `dest` will be null-terminated and will be filled whatever was successfully
241 ** written before the error
242 **
243 ** CAUTION: unlike the non-"Buf" variant, `max_writelen` is the number of bytes allowed to be written INCLUDING final '\0'
244 **
245 ** @param dest The buffer to write the escaped string to, if not `NULL`
246 ** @params max_writelen Maximum length of string to return (including final '\0'). No limit if set to `SIZE_ERROR`.
247 ** The function takes care of not reaching `max_writelen` in the middle of
248 ** an alias or encoded char by realizing there is not enough space and stopping
249 ** before writting it.
250 ** This means that the returned string may be shorter than `max_writelen` even
251 ** when it was limited by it
252 ** @returns
253 ** The number of bytes written to `dest` or that would have been written to `dest` if it was not `NULL`, or
254 ** SIZE_ERROR in case of error
255 */
