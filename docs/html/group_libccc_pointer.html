Detailed Documentation

This header defines the standard memory size and pointer integer types.

  • Standard library equivalent: (C) https://en.cppreference.com/w/c/types/integer

  • Standard library equivalent: (C) https://en.cppreference.com/w/c/types/limits

The types defined here are aliases for all the primitive integer types which have a varying (platform-dependent) amount of bits. These types are all used to deal with computer memory or adresses/pointers in some way.

These typedef s are here purely for nomenclature consistency reasons.

  • all primitive type aliases are prefixed with ‘t_’

  • all signed integer types have names that start with s, ie: sint

Typedefs

typedef size_t t_size

Primitive type: The memory byte size integer type

  • Standard library equivalent: (C) https://en.cppreference.com/w/c/types/size_t

See also:

  • Size_ToString()

  • Size_ToString_Hex()

  • Size_ToString_Pretty()

  • Size_FromString()

typedef ptrdiff_t t_ptrdiff

Primitive type: The pointer subtraction integer type

  • Standard library equivalent: (C) https://en.cppreference.com/w/c/types/ptrdiff_t

typedef intptr_t t_sintptr

Primitive type: Pointer-sized integer types

  • Standard library equivalent: (C) https://en.cppreference.com/w/cpp/types/integer Pointer-sized signed integer type

typedef uintptr_t t_uintptr

Pointer-sized unsigned integer type.

typedef intmax_t t_sintmax

Primitive type: Maximum-width integer types

  • Standard library equivalent: (C) https://en.cppreference.com/w/cpp/types/integer Maximum-width signed integer type

typedef uintmax_t t_uintmax

Maximum-width unsigned integer type.
