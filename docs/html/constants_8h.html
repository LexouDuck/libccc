Detailed Description

This header defines the common standard math constants and macro defines.

  • Standard library equivalent: (C) https://en.cppreference.com/w/c/numeric/math

This file holds various relevant mathematical constants. Many of these constants are used to speed up the fast-approximate versions of some fundamental algorithms and math functions.
