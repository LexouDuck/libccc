#include <logger.h>

struct logger
{
    // fields

    t_bool silence_logs;
    t_bool silence_errors;
    t_bool verbose;
    t_bool timestamp;
    t_bool obfuscated;
    t_bool append;
    e_logformat format;
    t_fd fd;
    t_char* path;
};

Detailed Documentation

This struct stores all the settings and internal state needed for a basic logging system.

Fields

t_bool silence_logs

If TRUE, then this logger will suppress “message”, “success” and “warning” logs.

t_bool silence_errors

If TRUE, then this logger will suppress “failure” and “error” logs.

t_bool verbose

If TRUE, then this logger will display “verbose” message logs (see Log_Verbose())

t_bool timestamp

If TRUE, the logger prints a timestamp at the beginning of each log line.

t_bool obfuscated

If TRUE, indicates that the logging messages should hide sensitive info, like passwords for instance.

t_bool append

If TRUE, logger will append text to the file if it already exists, rather than overwrite/clear it.

e_logformat format

Specifies the kind logging format for the text output.

t_fd fd

The file descriptor to which this logger outputs.

t_char* path

The file path (relative or absolute) for this logger output logfile.
