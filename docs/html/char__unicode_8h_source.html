460 ** @param out_symbolcount (optional) if not `NULL`, will be set to the total number of symbols.
461 ** If the entire string is valid, will be equivalent to the result of `CharUTF8_SymbolCount`

481 /*
482 ** @nonstd
483 **
484 ** Check if the entire utf8 string in composed of only valid utf8 sequences (@see CharUTF8_IsSeqValid) until the terminating
485 ** null character or until `n` bytes.
486 ** If `n` is reached in the middle of a UTF8 sequence, then `TRUE` is returned. However, it is only safe to read the
487 ** `out_bytecount` bytes of the string, anything after that will not have been checked (including the utf8 sequence
488 ** that could not be fully read because of the `n` bytes read limit)
489 **
490 **
491 ** @param str pointer to the beginning of the null-terminated uft8 string
492 ** @param out_symbolcount (optional) if not `NULL`, will be set to the total number of symbols.
493 ** If the entire string is valid, will be equivalent to the result of `CharUTF8_SymbolCount`
494 ** If the entire string is not valid, will be set to the number of valid symbols before
495 ** the failure point
496 ** if `str` is `NULL`, is set to SIZE_ERROR
497 ** @param out_bytecount (optional) if not `NULL`, will be set to the total number of byte.
498 ** If the entire string is valid, will be equivalent to the result of `String_Length`
499 ** If the entire string is not valid, will be set to the number of valid bytes before
500 ** the failure point
501 ** Even though the parameter is optional for API consistency, only `out_bytes` can be
502 ** safely read when the function returns `TRUE`, so it is highly adviced to use this
503 ** out parameter
504 ** if `str` is `NULL`, is set to SIZE_ERROR
505 **
506 ** @returns
507 ** `TRUE` if the given utf8 string is entirely made up of valid UTF8 sequences until the terminating '\0',
508 ** otherwise returns `FALSE`
509 **
510 */
