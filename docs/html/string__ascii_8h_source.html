1293 ** written before the error

1295 ** CAUTION: unlike the non-"Buf" variant, `max_writelen` is the number of bytes allowed to be written INCLUDING final '\0'

1297 ** @param dest The buffer to write the escaped string to, if not `NULL`
1298 ** @params max_writelen Maximum length of string to return (including final '\0'). No limit if set to `SIZE_ERROR`.

1302 ** This means that the returned string may be shorter than `max_writelen` even
1303 ** when it was limited by it
1304 ** @returns
