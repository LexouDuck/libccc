Detailed Documentation

This header defines the common standard text string manipulation functions.

  • Standard library equivalent: (C) https://en.cppreference.com/w/c/string/byte#String_manipulation

Global Variables

HEADER_CPP typedef t_char* p_str

Type definition for a string of characters.

This typedef is only provided for certain niche purposes, since it is typically not very good practice in C to hide pointers within typedef types.

Global Functions

t_char* String_New_C(t_size n, t_char c)

Returns a newly allocated region of c -filled memory, which is n + 1 bytes long.

  • This function is not defined in the C ISO standard.

Allocates a new region of memory of the specified size n + 1, the last byte being used for the string null terminator character 0’`.

Parameters:

n

The length of the string to allocate (not including null-terminator)

c

The character value to fill each byte with

Returns:

A newly allocated string which is n + 1 bytes in length, and has every char set to c, plus the last char set to 0’`.
