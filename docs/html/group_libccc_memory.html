Detailed Documentation

This header defines the common standard memory manipulation functions.

  • Standard library equivalent: (C) https://en.cppreference.com/w/c/memory

Global Variables

HEADER_CPP typedef void* p_any

Type definition for a pointer

This typedef is only provided for certain niche purposes, since it is typically not very good practice in C to hide pointers within typedef types.

See also:

Global Functions

void* Memory_New_C(t_size size, char c)

Returns a newly allocated memory area which is size bytes long, with every byte set to c.

  • This function is not defined in the C ISO standard.

Allocates size bytes in memory, setting each byte of this newly allocated memory space to c. Returns the pointer at which said bytes were allocated, or NULL if the memory could not be allocated.

Returns:

The pointer to newly allocated region, or NULL if an error occurred (ie: the memory could not be allocated for whatever reason).

void* Memory_Allocate(t_size size)

Returns a pointer to a newly allocated memory area which is size bytes long, uninitialized.

  • Standard library equivalent: (C89) https://en.cppreference.com/w/c/memory/malloc

Allocates size bytes in memory, returning the pointer to allocated region. NB: This function only allocates a new memory region and does nothign more: This means that there may be garbage leftover values in the newly allocated region. It generally recommended to use Memory_New() instead, to avoid garbage memory issues.

Returns:

The pointer to newly allocated region, or NULL if an error occurred (ie: the memory could not be allocated for whatever reason).

void* Memory_Reallocate(void* ptr, t_size size)

Reallocates the given area of memory ptr to be size bytes long, returning the new (or same) pointer.

  • Standard library equivalent: (C89) https://en.cppreference.com/w/c/memory/realloc

This function will either reallocate in-place (ie: not changing the pointer), or, if that is not possible, it will allocate the needed size to a new region.

Returns:

The pointer to newly allocated region, or NULL if an error occurred (ie: the memory could not be allocated for whatever reason). NOTE: If the requested region is not too different in size, then the return value may be the exact same pointer as the given ptr argument.

void Memory_Deallocate(void* ptr)

Frees the previously allocated memory area at *ptr.

  • Standard library equivalent: (C89) https://en.cppreference.com/w/c/memory/free

Deallocates the area of memory pointed to by ptr, assuming it was previously allocated by a call to Memory_Alloc() or Memory_New()

void Memory_Delete(void** a_ptr)

Frees the allocated memory at **a_ptr, and sets *a_ptr to NULL.

  • This function is not defined in the C ISO standard.

Frees the allocated memory at **a_ptr, and sets *a_ptr to NULL.

Parameters:

a_ptr

The address of the pointer to free and NULL ify

 
void Memory_Set(void* ptr, t_u8 byte, t_size n)

Sets n bytes of memory, starting at ptr, to the given value byte.

  • Standard library equivalent: (C89) https://en.cppreference.com/w/c/string/byte/memset

Sets n bytes of memory with the given 8-bit value byte (taking only the first 8 bits of this value and writing it byte-per-byte), starting at ptr.

void Memory_Clear(void* ptr, t_size n)

Sets n bytes of memory to 0, starting at ptr.

  • Standard library equivalent: (BSD) https://linux.die.net/man/3/bzero

Sets n bytes of memory to 0, starting at ptr. (equivalent to bzero)

void* Memory_Copy(void* dest, void const* src, t_size n)

Copies n bytes of data from src into dest, and returns dest.

  • Standard library equivalent: (C89) https://en.cppreference.com/w/c/string/byte/memcpy

Copies n bytes of memory from src to dest, and returns dest.

void* Memory_Copy_C(void* dest, void const* src, t_u8 byte, t_size n)

Copies n bytes of data from src into dest, and returns dest (stops upon encountering byte).

  • Standard library equivalent: (C23) https://en.cppreference.com/w/c/string/byte/memccpy

Copies at most n bytes of memory from src to dest, stopping after the first occurence of a byte equal to byte, and returns a pointer to (byte + 1) in dest, or NULL.

void* Memory_Move(void* dest, void const* src, t_size n)

Moves n bytes of data from src into dest, and returns dest.

  • Standard library equivalent: (C89) https://en.cppreference.com/w/c/string/byte/memmove

Reads n bytes of memory from src, and then writes all of those bytes to dest after having read everything. This function is useful if the src and dest buffers overlap.

void* Memory_Duplicate(void const* ptr, t_size n)

Allocates a memory region which is n bytes long and has the same content as ptr.

  • This function is not defined in the C ISO standard.

Returns a newly allocated memory area which is a copy of the given memory area ptr (or NULL if the required memory could not be allocated, or if ptr == NULL` or n == 0`).

void* Memory_Join(void const* ptr1, t_size length1, void const* ptr2, t_size length2)

Return a newly allocated memory region, by concatenating ptr1 and ptr2

  • This function is not defined in the C ISO standard.

void* Memory_Append(void** a_dest, t_size dest_length, void const* src, t_size src_length)

Return a newly allocated memory region, by concatenating *a_dest (deletes old) and src

  • This function is not defined in the C ISO standard.

void* Memory_Prepend(void const* src, t_size src_length, void** a_dest, t_size dest_length)

Return a newly allocated memory region, by concatenating src and *a_dest (deletes old)

  • This function is not defined in the C ISO standard.

void* Memory_Merge(void** a_ptr1, t_size length1, void** a_ptr2, t_size length2)

Return a newly allocated memory region, by concatenating *a_ptr1 (deletes old) and *a_ptr2 (deletes old)

  • This function is not defined in the C ISO standard.

void* Memory_Find(void const* ptr, t_u8 byte, t_size n)

Finds the first occurence of the given byte value in the first n bytes at ptr.

  • Standard library equivalent: (C89) https://en.cppreference.com/w/c/string/byte/memchr

Returns a pointer to the first occurrence of a byte which is equal to the given byte value, or NULL if no matching byte was encountered. Begins the search at ptr, and reads through n bytes of memory.

void Memory_Replace(void* ptr, t_u8 old, t_u8 new, t_size n)

Replaces every old byte with new, in the n byte long memory region starting at ptr.

  • This function is not defined in the C ISO standard.

Iterates through n bytes of memory starting at ptr, replacing every byte equal to old with the value new instead.

t_sint Memory_Compare(void const* ptr1, void const* ptr2, t_size n)

Compares n bytes of memory in ptr1 and ptr2 : returns the first difference of chars encountered, or zero, if the bytes all match.

  • Standard library equivalent: (C89) https://en.cppreference.com/w/c/string/byte/memcmp

Compares n bytes of memory between ptr1 and ptr2.

Returns:

The first difference between bytes encountered: (ptr1[i] - ptr2[i]), or 0 if the contents of ptr1 and ptr2 match.

t_bool Memory_Equals(void const* ptr1, void const* ptr2, t_size n)

Checks n bytes of memory in ptr1 and ptr2 for equality

  • This function is not defined in the C ISO standard.

Compares n bytes of memory between ptr1 and ptr2.

Returns:

TRUE if the n first bytes of ptr1 and ptr2 are identical. Otherwise, returns FALSE.

t_bool Memory_Swap(void* ptr1, void* ptr2, t_size size)

Swaps size bytes of memory between ptr1 and ptr2

  • This function is not defined in the C ISO standard.

Swaps the memory content pointed to by ptr1 and ptr2, copying exactly size bytes of data between the two. (XOR swap method)

Returns:

FALSE if the swap was successful, or TRUE otherwise.

Get a subsection of n bits from the data in ptr, starting at bit index bit.

  • This function is not defined in the C ISO standard.

t_s8 Memory_GetMostSignificantBit(t_uintmax value)

Get the bit index of the most signficant 1 -bit of the given value

  • This function is not defined in the C ISO standard., similar to the builtin clz() function

Returns:

The bit index of the highest bit which is set to 1 in the given value. If value == 0 (all bits set to zero), returns -1

t_s8 Memory_GetLeastSignificantBit(t_uintmax value)

Get the bit index of the least signficant 1 -bit of the given value

  • This function is not defined in the C ISO standard., similar to the builtin clz() function

Returns:

The bit index of the lowest bit which is set to 1 in the given value. If value == 0 (all bits set to zero), returns -1
