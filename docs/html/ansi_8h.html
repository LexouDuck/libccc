◆ ANSI

#define ANSI (   X)    "\x1B["X"m"

ANSI text output display escape code macros.

These are some useful string literals for commandline output colors. Can be used with any of the 'IO_Output_*' and 'IO_Write_*' functions.

(Learn more here)[https://misc.flogisoft.com/bash/tip_colors_and_formatting]
