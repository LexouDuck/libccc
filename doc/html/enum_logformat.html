enum logformat¶

Overview¶

This enum specifies a specific text formatting wrapper. More…

#include <logger.h>

enum logformat
{
    LOGFORMAT_ANSI = 0,
    LOGFORMAT_TEXT = 1,
    LOGFORMAT_JSON = 2,
    LOGFORMAT_XML  = 3,
};

Detailed Documentation¶

This enum specifies a specific text formatting wrapper.

Enum Values¶

LOGFORMAT_TEXT

Prints in a format that is friendly for terminals (ANSI colors)

LOGFORMAT_JSON

Prints in a format that is friendly for raw text (same as above, but without ANSI color codes)

LOGFORMAT_XML

Prints in a format that can be easily parsed by a JSON parser.
