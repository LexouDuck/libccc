Detailed Documentation¶

This header defines the common standard math functions and macro defines.

  • C ISO Standard: https://en.cppreference.com/w/c/numeric/math

Macros¶

#define ABS(X)  ((X)<0 ? -(X) :(X))

Expands to the absolute value of x (ie: gives the positive equivalent of x)

  • C ISO Standard: https://en.cppreference.com/w/c/numeric/math/abs

  • C ISO Standard: https://en.cppreference.com/w/c/numeric/math/fabs

  • Math: \({|x|}\)

#define MAX(X, Y)   ((X)<(Y) ?(Y) :(X))

Expands to the maximum value between x and y (the larger of the two)

  • C ISO Standard: https://en.cppreference.com/w/c/numeric/math/fmax

#define MIN(X, Y)   ((X)<(Y) ?(X) :(Y))

Expands to the minimum value between x and y (the smaller of the two)

  • C ISO Standard: https://en.cppreference.com/w/c/numeric/math/fmin
