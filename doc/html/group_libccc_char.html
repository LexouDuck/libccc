Detailed Documentation¶

This header defines all the functions/sets related to single ASCII chars.

  • C ISO Standard: https://en.cppreference.com/w/c/language/ascii

  • https://en.wikipedia.org/wiki/UTF-8

  • https://en.wikipedia.org/wiki/UTF-16

  • https://en.wikipedia.org/wiki/UTF-32

Typedefs¶

typedef char t_char

Primitive type: char, ie: a text character (1 byte: ANSI/ASCII)

  • C ISO Standard: https://en.cppreference.com/w/c/string/byte

typedef char t_utf8

Primitive type: Unicode UTF-8 character (variable length: 1 to 4 bytes)

  • C ISO Standard: https://en.cppreference.com/w/c/string/multibyte

typedef uint_least16_t t_utf16

Primitive type: Unicode UTF-16 character (variable length: 2 or 4 bytes)

  • C ISO Standard: https://en.cppreference.com/w/c/string/multibyte/char16_t

  • C ISO Standard: https://en.cppreference.com/w/c/string/multibyte

typedef uint_least32_t t_utf32

Primitive type: Unicode UTF-32 character code point value (fixed length: 4 bytes)

  • C ISO Standard: https://en.cppreference.com/w/c/string/multibyte/char32_t

  • C ISO Standard: https://en.cppreference.com/w/c/string/wide
