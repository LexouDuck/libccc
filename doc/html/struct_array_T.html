Detailed Documentation¶

This struct holds an array of items which can be of any type.

The ‘s_array’ struct holds a void* pointer to the array of items, the size of a single item in the array, and the total amount of items in this array. As such, a s_array can hold any number of items, but they must all share the same type.

Fields¶

t_uint length

The amount of elements in the ‘items’ array.

T* items

The pointer to the array (items can be of any one type)
