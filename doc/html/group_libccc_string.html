Detailed Documentation¶

This header defines the common standard string (char*) manipulation functions.

  • C ISO Standard: https://en.cppreference.com/w/c/string/byte#String_manipulation

Global Functions¶

char* String_New(t_size n)

Returns a newly allocated region of zero-filled memory, which is n + 1 bytes long.

Allocates a new region of memory of the specified size n + 1, the last byte being used for the string null terminator character ‘0’.

Parameters:

n

The length of the string to allocate

Returns:

A new string which is n + 1 bytes in length, and has every char set to ‘0’.

char* String_New_C(t_size n, char c)

Returns a newly allocated region of c -filled memory, which is n + 1 bytes long.

Allocates a new region of memory of the specified size n + 1, the last byte being used for the string null terminator character ‘0’.

Parameters:

n

The length of the string to allocate

c

The character value to fill each byte with

Returns:

A new string which is n + 1 bytes in length, and has every char set to c, plus the last char set to ‘0’.

void String_Set(char* str, char c)

Sets every char of the given string str to c, only stopping upon reading a ‘0’ string terminator character.

Fills the given string str with the character c

Parameters:

str

The string to fill

c

The character value to fill with

 
void String_Clear(char* str)

Clears the given string str, setting each character of str to ‘0’.

void String_Delete(char** a_str)

Deletes the string pointed to by ptr, freeing the associated memory.

char* String_Duplicate(char const* str)

Returns a newly allocated string which is a copy of the given string str, (or NULL required memory could not be allocated).

char* String_Duplicate_N(char const* str, t_size n)

Returns a newly allocated string which is a copy of the given string str, (or NULL required memory could not be allocated), copying at most n characters.

char* String_Duplicate_Char(char const* str, char const c)

Returns a newly allocated string which is a copy of the given string str, (or NULL required memory could not be allocated), stopping at the first occurence of c.

char* String_Copy(char* dest, char const* src)

Copies the given string src into dest (null-terminator included), and returns dest (no allocation is performed).

char* String_Copy_N(char* dest, char const* src, t_size n)

Copies the first n characters of the given string src into dest, and returns dest

Copies the first n characters of the given string src into dest, and returns dest (no allocation is performed).

t_size String_Copy_L(char* dest, char const* src, t_size size)

Copies the given string src into dest, null-terminating the result and returning its length.

Copies the given string src into dest, null-terminating the result (the ‘0’ is placed at: (dest + size - 1)), and returns the resulting size of dest.

char* String_Concat(char* dest, char const* src)

Concatenates the given string src to the end of dest, (removing dest’s ‘0’ terminator) and returns dest.

char* String_Concat_N(char* dest, char const* src, t_size n)

Concatenates the first n characters of the given string src to the end of dest, and returns dest.

t_size String_Concat_L(char* dest, char const* src, t_size size)

Concatenates characters of the given string src to the end of dest, null-terminating the result (places the ‘0’ at dest + size - 1), and returns the resulting size of dest.

t_size String_Length(char const* str)

int String_Compare(char const* str1, char const* str2)

int String_Compare_N(char const* str1, char const* str2, t_size n)

int String_Compare_IgnoreCase(char const* str1, char const* str2)

char* String_ToEscape(char const* str)

Returns a new null-terminated string which is a copy of str, in which all non-printable characters found in str are made into printable ANSI-C ‘xHH’ escape sequences (where “H”s are uppercase hex digits).

char* String_Replace_Char(char const* str, char const char_old, char const char_new)

Returns a new null-terminated string which is a copy of str, in which every occurence of an old char is replaced with a new char.

char* String_Replace_Charset(char const* str, char const* charset_old, char const* charset_new)

Returns a new null-terminated string which is a copy of str, in which every char in the old charset is replaced by the corresponding character in the new charset (at the same index).

char* String_Replace_String(char const* str, char const* str_old, char const* str_new)

Returns a new null-terminated string which is a copy of str, in which every occurence of the string old is replaced with new. NB: see stringarray.h -> String_Split_String for extra notes on bevahior.

char* String_Join(char const* str1, char const* str2)

Returns a new null-terminated string which is the result of the concatenation of the two given strings str1 and str2.

char* String_Merge(char** a_s1, char** a_s2)

Returns a newly allocated string which is the concatenation of s1 and s2. Deletes both inputs from memory. Also, returns the result.

char* String_Append(char** dest, char const* src)

Returns a newly allocated string which is the concatenation of dest and src. Deletes dest, replacing it by the result. Also, returns the result.

char* String_Prepend(char const* src, char** dest)

Returns a newly allocated string which is the concatenation of src and dest. Deletes dest, replacing it by the result. Also, returns the result.

char* String_Insert_InPlace(char** dest, char const* src, t_u32 index)

Inserts the string src at index index in dest; deletes dest and replaces it by the result. Also returns the result.

void String_Replace_Char_InPlace(char* str, char const char_old, char const char_new)

Replaces every occurence of old by new in str.

void String_Replace_Charset_InPlace(char* str, char const* charset_old, char const* charset_new)

Replaces old charset by new charset in str. Works like the bash function tr. Substitutes old[i] by new[i] in str. Fails silently if strlen is different for old and new, or if a character is repeated in old (only one image for every antecedent), or if old or new are the empty string.

void String_Replace_String_InPlace(char** a_str, char const* str_old, char const* str_new)

Replaces every occurence of the query old by the string new in str. NB: see stringarray.h -> c_strsplit_str for extra notes on bevahior.

char* String_Map_InPlace(char** a_str, char(*)(char) f)

Changes the content of ‘*a_str’ by applying f to each of its chars. Deletes a_str, and sets the result in it instead. Also returns the result.

char* String_Trim(char const* str, char const* charset)

Returns a new string from str in which all leading and trailing whitespace characters (‘ ‘, ‘t’, ‘n’, ‘r’, ‘v’, ‘f’) have been removed.

char* String_Trim_L(char const* str, char const* charset)

Returns a new string from str in which all leading characters present in charset have been removed.

char* String_Trim_R(char const* str, char const* charset)

Returns a new string from str in which all trailing characters present in charset have been removed.

char* String_Pad(char const* str, char c, t_size length)

Returns a new null-terminated string duplicate of str which is length characters long, by padding it with c chars on both sides if the desired length is larger than str.

char* String_Pad_L(char const* str, char c, t_size length)

Returns a new null-terminated string duplicate of str which is length characters long, by padding it with c chars on its left side if the desired length is larger than str.

char* String_Pad_R(char const* str, char c, t_size length)

Returns a new null-terminated string duplicate of str which is length characters long, by padding it with c chars on its right side if the desired length is larger than str.

char* String_ToPrintableString(char const* str)

Returns a new null-terminated string where every non-printable character of str is replaced by either its normal escape sequence (if available) or a ‘xFF’-type byte escape sequence if no simple short escape sequence exists.

char* String_Reverse(char const* str)

Returns a newly allocated string which is a copy of the given string str, but in reverse order (except for the 0 terminator, obviously).

char* String_Insert(char const* dest, char const* src, t_size offset)

Returns a reallocated version of the given string dest, in which the string str has been inserted at the index offset.

char* String_Sub(char const* str, t_size index, t_size n)

Returns a new null-terminated string which is a subsection of str, starting at char index index and copying n characters.

void String_Iterate(char* str, void(*)(char*c) f)

Iterates upon each character of the given string str, applying the given function f to each of its characters.

void String_Iterate_I(char* str, void(*)(t_size index, char*c) f)

Iterates upon each character of the given string str, applying the function f to each of its chars (with index information).

char* String_Map(char const* str, char(*)(char c) f)

Creates a new null-terminated string by iterating upon the string str, applying the function f to each of its chars.

char* String_Map_I(char const* str, char(*)(t_size index, char c) f)

Creates a new null-terminated string by iterating upon the string str, applying the function f to each of its chars (with index information).
