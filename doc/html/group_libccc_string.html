Detailed Documentation¶

This header defines the common standard string (t_char*) manipulation functions.

  • Standard library equivalent: (https://en.cppreference.com/w/c/string/byte#String_manipulation) C

Global Functions¶

t_size String_Print(t_char** dest, t_char const* str, t_size n, t_char const* charset_extra)

Creates a new string from str, replacing special characters with appropriate escape-sequences.

  • This function is not defined in the C ISO standard.

Returns a new null-terminated string where every non-printable character of str is replaced by either its normal escape sequence (if available) or a ‘xFF’-type byte/unicode escape sequence if no simple short escape sequence exists. Here is the list of characters which will be escaped by default:

  • \\ \ (a single backslash, escaping the escape character)

  • &rsquo; Apostrophe - ” Double quotes - 0 Null character (string terminator) - ** Bell/Alert/Audible - **** Backspace - t Tab character -

    Line feed - v Vertical tab - f Form-feed - r Carriage return - ** Escape - x?? Byte value, written as hexadecimal

  • TODO u???? UTF-8 multi-byte character, written as a hexadecimal code point (Unicode: U+????)

  • TODO U????????` UTF-8 multi-byte character, written as a hexadecimal code point (Unicode: U+????????)

Parameters:

dest

The destination string

str

The string to print/duplicate, with escape-sequences

n

The maximum amount of characters to print (infinite if 0 is given)

charset_extra

A string containing any extra characters which should be escaped (ie: any char in charset_extra will become a \x?? byte escape sequence)

Returns:

The amount of characters parsed from the given str.

t_size String_Parse(t_char** dest, t_char const* str, t_size n, t_bool any_escape)

Creates a new string from str, replacing escape-sequences with their corresponding char value.

  • This function is not defined in the C ISO standard.

Returns a new null-terminated string where every valid backslash escape sequence is converted to its corresponding string byte. Here is the list of character escape sequences which will be properly parsed:

  • \\ \ (a single backslash, escaping the escape character)

  • &rsquo; Apostrophe - ” Double quotes - 0 Null character (string terminator) - ** Bell/Alert/Audible - **** Backspace - t Tab character -

    Line feed - v Vertical tab - f Form-feed - r Carriage return - ** Escape - x?? Byte value, written as hexadecimal - u???? UTF-8 multi-byte character, written as a hexadecimal code point (Unicode: U+????) - U????????` UTF-8 multi-byte character, written as a hexadecimal code point (Unicode: U+????????) Any other backslash-ed character will simply resolve to itself (ie: removing the preceding backslash)

Parameters:

dest

The destination string

str

The string to duplicate, while resolving all escape-sequences to their corresponding char

n

The maximum amount of characters to parse (infinite if 0 is given)

any_escape

If TRUE, every backslash will be understood as an escape character (ie: any escape sequance will function, with any char after the '`)

Returns:

The amount of characters parsed from the given str.

t_char* String_New(t_size n)

Returns a new string which is n + 1 bytes long (to account for the null-terminator).

  • This function is not defined in the C ISO standard.

Allocates a new region of memory of the specified size n + 1, the last byte being used for the string null terminator character 0’`.

Parameters:

n

The length of the string to allocate (not including null-terminator)

Returns:

A newly allocated region of zero-filled memory which is n + 1 bytes in length, and has every char set to 0’`.

t_char* String_New_C(t_size n, t_char c)

Returns a newly allocated region of c -filled memory, which is n + 1 bytes long.

  • This function is not defined in the C ISO standard.

Allocates a new region of memory of the specified size n + 1, the last byte being used for the string null terminator character 0’`.

Parameters:

n

The length of the string to allocate (not including null-terminator)

c

The character value to fill each byte with

Returns:

A newly allocated string which is n + 1 bytes in length, and has every char set to c, plus the last char set to 0’`.
