Macros¶

#define UTF16_BIAS  0x10000

These macros are used to encode/decode UTF-16 strings.

The UTF-16 byte order mark: FF FE 00 00 (Little-Endian)

#define UTF8_1BYTE  0x00000080

Beneath this code point, UTF-8 encodes this as a 1-byte character.

These macros are used to encode/decode UTF-8 strings.

#define UTF8_2BYTE  0x00000800

Beneath this code point, UTF-8 encodes this as a 2-byte character.

#define UTF8_3BYTE  0x00010000

Beneath this code point, UTF-8 encodes this as a 3-byte character.

#define UTF8_4BYTE  0x0010FFFF

Beneath this code point, UTF-8 encodes this as a 4-byte character

#define UTF8_BOM    "\xEF\xBB\xBF"

The Unicode byte order mark (https://en.wikipedia.org/wiki/Byte_order_mark) macros. A byte order mark is used at the beginning of a string, to indicate that the string is indeed a Unicode string, the type of Unicode encoding, and the byte endian-ness used. The UTF-8 byte order mark: EF BB BF
